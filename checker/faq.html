MAZE-ARENA NFT is an experimental multi-token (ERC1155) contract (non-fungible and fungible tokens in a single contract).
ERC1155 tokens have 0 decimal places.


To view and send tokens you must have MetaMask wallet configured for SmartBCH (Bitcoin Cash side-chain). Refresh the ARENA NFT Checker page if it is not connected to MetaMask.
Type/paste token ID and click/tap buttons (you may need to double-click).
To view custom ERC1155 tokens you must provide token ID, contract address and base image url (ask the token creator about it).

Network settings for MetaMask:
Network name: SmartBCH
RPC URL: https://smartbch.fountainhead.cash/mainnet or https://smartbch.greyh.at
Chain ID: 10000
Currency Symbol: BCH
Explorer: https://smartscan.cash
